# Per-file shorthands
- use Abbr package for written shorthands and abbreviations
- write custom show-rules for others

# Custom Shorthands
## Math symbols
- iff = <=>
- if = =>
- imp = if
- follows = if
- to = ->
- dto = |->
- neq = math.eq.not

## Utils
- to-do = big todo break
- hx = highlight text
- tx = text colour
 
## Abbr
- a = abbr.a
- apl = abbr.pla
- al = abbr.l
- apll = abbr.pll
